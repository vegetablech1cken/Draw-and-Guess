# 🎮 你画我猜游戏 - 快速开始指南

## 📦 安装依赖

```bash
# 安装运行时依赖
pip install -r requirements.txt
```

## 🚀 启动游戏

### 方法一：使用提供的客户端

1. **启动服务器**（在第一个终端）：
```bash
python src/server/main.py
```

2. **启动客户端**（在第二个终端）：
```bash
python src/client/game_client.py
```

3. **输入玩家名称并开始游戏**

### 方法二：直接运行（推荐用于测试）

启动服务器后，可以启动多个客户端进行测试：

```bash
# 终端1：启动服务器
python src/server/main.py

# 终端2：启动第一个客户端
python src/client/game_client.py

# 终端3：启动第二个客户端（可选，用于多人测试）
python src/client/game_client.py
```

## 🎨 游戏玩法

### 大厅阶段
1. 连接到服务器后，你会进入游戏大厅
2. 等待其他玩家加入（至少需要2名玩家）
3. 点击"开始游戏"按钮开始游戏

### 游戏阶段

#### 如果你是画家：
- 在画布上拖动鼠标绘制图案
- 点击颜色按钮选择不同的颜色
- 点击"清空画布"清除所有内容
- 尽量让其他玩家猜出你画的词语

#### 如果你是猜测者：
- 观察画家的画作
- 在输入框中输入你的猜测
- 按回车提交答案
- 猜对了会获得分数！

## ⌨️ 操作说明

### 画布操作
- **鼠标左键拖动**：绘图
- **点击颜色方块**：更换画笔颜色
- **清空画布按钮**：清除所有绘图（仅画家可用）

### 猜测和聊天
- **输入框**：输入文字
- **按回车**：发送猜测或聊天消息

## 🏆 计分规则

- 越早猜对得分越高
- 第一个猜对：100分
- 第二个猜对：90分
- 第三个猜对：80分
- ...以此类推，最少10分

## 🔧 配置说明

默认配置（在 `src/shared/constants.py` 中）：
- **服务器地址**：127.0.0.1（本地）
- **服务器端口**：5555
- **最大玩家数**：8人
- **窗口大小**：1280x720

如需修改，请编辑 `src/shared/constants.py` 文件。

## ⚠️ 注意事项

1. **网络连接**：确保服务器和客户端在同一网络环境下，或服务器端口已开放
2. **Python版本**：需要 Python 3.8 或更高版本
3. **pygame**：首次运行需要安装 pygame 库
4. **多人游戏**：至少需要2名玩家才能开始游戏
5. **词库**：游戏词语来自 `data/words.txt` 文件，可自行添加

## 🐛 常见问题

### 1. 无法连接到服务器
- 检查服务器是否已启动
- 确认防火墙设置
- 验证IP地址和端口配置

### 2. 画布不响应
- 确认你是当前回合的画家
- 检查鼠标是否在画布区域内

### 3. 无法输入文字
- 点击输入框激活它
- 确认输入法状态

## 🔄 游戏流程

1. **启动服务器** → 监听客户端连接
2. **客户端连接** → 加入默认房间
3. **等待玩家** → 至少2人
4. **开始游戏** → 随机选择画家和词语
5. **画家绘图** → 实时同步到所有玩家
6. **玩家猜测** → 正确则得分
7. **下一轮** → 轮流当画家
